Movie Api (myFlix back-end)

This API allows a users to access a movie database that contains image, synopsis and information about the director and movie genre. It has also endpoints for the user to register/deregister and log in/out update user profile and add movies to a list of favorites. This app was built by using node.js, Express, MongoDB, mongoose, and Postman.

Myflix (React)

This is the client-side of my first full stack developer program. MyFlix is a single page app that displays information about tech related movies. Registered users can create a list of their favorite movies. In the development process React, React Redux, React bootstrap, axios and Parcel were used.
